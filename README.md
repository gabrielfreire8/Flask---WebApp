# Flask---WebApp
Desenvolvi uma base sólida em Flask, entendendo por que ele é considerado um microframework e quais são suas vantagens no desenvolvimento de aplicações web em Python.
Ao longo do desenvolvimento implementei alguns conceitos do zero, como: 
- Criação e configuração de uma aplicação Flask

- Definição de rotas utilizando @app.route

- Integração entre Python e HTML com render_template e Jinja2

- Uso de lógica Python dentro dos templates HTML

- Aplicação de Orientação a Objetos no projeto

- Manipulação de formulários e tratamento de requisições GET e POST

- Utilização do debugger para agilizar o desenvolvimento

- Estilização da aplicação com Bootstrap e organização de arquivos no diretório static

- Uso de herança de templates para evitar código duplicado

- Geração dinâmica de URLs com url_for

- Gerenciamento de sessões com session (login e persistência temporária de dados)

- Uso de query strings para troca de informações entre páginas
